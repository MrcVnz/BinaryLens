BinaryLens update notes
=======================

Highlights
- improved embedded payload calibration for clean archives and compressed/containerized data
- validated embedded pe detection instead of trusting any loose mz marker
- softened low-confidence wording when payload corroboration is weak
- added disposition, signal reliability, corroboration count, and compressed-noise hints to embedded payload reporting
- kept VirusTotal configuration compatible with config.json and environment variables
- refreshed README documentation in English and pt-BR

Recommended verification after build
- test a known-clean zip archive
- test a signed installer/bootstrapper
- test a suspicious file or packed sample
- confirm VirusTotal works with your local key
