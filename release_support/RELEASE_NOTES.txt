BinaryLens release workflow

1. run release_support\make_portable_release.bat
2. confirm release\BinaryLens-Portable contains BinaryLensQt.exe, qt dlls, plugins, rules, plugins, and config\config.json
3. zip release\BinaryLens-Portable for the portable release
4. open release_support\BinaryLens.iss in Inno Setup and compile it
5. publish the generated installer and the portable zip as GitHub release assets
