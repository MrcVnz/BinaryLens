ASM build notes
===============

required asm files
------------------
- add asm/pattern_scan_x64.asm as a Microsoft Macro Assembler source
- add asm/entrypoint_profile_x64.asm as a Microsoft Macro Assembler source

why both matter
---------------
- pattern_scan_x64.asm exports BL_FindPatternMasked_Asm
- entrypoint_profile_x64.asm exports BL_ProfileEntrypointStub_Asm

cmake reminder
--------------
When using CMake with MSVC x64, enable ASM_MASM and attach both files to the same target.
See BUILD_INTEGRATION_NOTES.txt and cmake/asm_target_sources_snippet.cmake for a ready snippet.
